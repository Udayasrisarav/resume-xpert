# ResumeXpert — AI Resume Analyzer

An AI-powered web application that helps job seekers evaluate their resumes with **ATS scoring, parsing, and actionable feedback**. Built with **React.js** and deployed on **Vercel**, ResumXpert combines modern web technologies with practical career insights.

## 🚀 Features
- 📄 Upload resumes in PDF format for instant analysis  
- 🧠 AI-driven **ATS scoring & feedback visualization**  
- 🔍 Resume parsing with structured insights (Summary, ATS Score, Details)  
- ☁️ Secure data handling using **Puter KV storage** + cloud APIs  
- 🎨 Responsive UI with **TailwindCSS** and modular components  
- 📑 Dynamic PDF rendering & image previews  

## 🛠️ Tech Stack
- **Frontend:** React.js, TailwindCSS, Vite  
- **Backend/Server:** React Router (SSR), Node.js  
- **Deployment:** Vercel  
- **Storage & APIs:** Puter KV, Cloud APIs
